# Штука для упрощения в переводе

Special for Ярик

## Софт

Для работы нужен установленный `node.js`. Скачать с оффициального сайта: <https://nodejs.org/en/download>

## Запуск

Установка пакетов
```sh
npm i
```

Запуск клиента
```sh
npm run client
```

Запуск сервера
```sh
npm run server
```

Клиент и сервер запускать в отдельных окнах консоли
